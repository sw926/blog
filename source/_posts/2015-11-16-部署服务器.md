通过ssh登录服务器
--------
```sh ssh登录
ssh username@server

```
配置本地hosts
--------
在/etc/hosts加入
```
{服务器id} server
```
例如
```
123.123.123.123 server
```
配置ssh免输入密码
--------
```sh
cd ~/.ssh
ssh-keygen -t rsa
```
上传到服务器
```sh
scp ~/.ssh/id_rsa.pub username@server:~/.ssh/id_rsa.pub
```
在服务器操作
```sh
ssh username@server
cd ~/.ssh
cat id_rsa.pub >> authorized_keys
rm id_rsa.pub
```
安装MySql
--------
```sh
apt-get update
sudo apt-get install mysql-server
```
安装Ruby
--------
安装 RVM
```sh
$ curl -L https://get.rvm.io | bash
```
重新登录下服务器，检查rvm是否安装成功
```sh
rvm info
```
安装ruby
```sh
rvm install 2.2.2
rvm use 2.2.2 --default
```
