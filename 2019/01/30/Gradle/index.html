Project

生命周期

Project 和 build.gradle 是一对一的关系，Gradle 会在初始化阶段为每个 build.gradle 文件创建一个 Project 实例，过程如下：

  • 创建一个 Settings 实例
  • 执行 settings.gradle 脚本，对 Settings 对象进行配置
  • 使用配置后的 Settings 对象创建工程目录
  • 最后执行每个 build.gradle 脚本
